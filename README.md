# Customer Support Chatbot (Dialogflow + Telegram)

## Overview
A conversational AI chatbot built using Dialogflow and integrated with Telegram
to automate customer support queries.

## Features
- Intent-based NLP responses
- Telegram integration
- Fallback handling
- FAQ automation

## Tech Stack
- Dialogflow ES
- Telegram Bot API

## Setup Instructions
1. Create Dialogflow agent
2. Import agent ZIP
3. Create Telegram bot using BotFather
4. Paste token in Dialogflow → Integrations → Telegram

## Demo
Open Telegram and search: @your_bot_name
